# House Price Prediction

## Project Overview
Developed a machine learning regression model to predict housing prices based on demographic and housing features.

## Dataset
USA Housing Dataset

## Tasks Performed
- Exploratory Data Analysis
- Feature Selection
- Data Preprocessing
- Linear Regression Model Training
- Model Evaluation

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- scikit-learn

## Outcome
Successfully built a regression model for estimating house prices.
