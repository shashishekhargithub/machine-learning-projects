# Loan Default Prediction

## Project Overview
Built a Random Forest classification model to predict whether borrowers would repay their loans.

## Dataset
Lending Club Loan Dataset

## Tasks Performed
- Data Cleaning
- Exploratory Data Analysis
- Feature Engineering
- Random Forest Model Training
- Performance Evaluation

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- scikit-learn

## Outcome
Successfully classified loan repayment outcomes using ensemble learning techniques.
