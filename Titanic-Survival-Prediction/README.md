# Titanic Survival Prediction

## Project Overview
Built a machine learning classification model to predict passenger survival on the Titanic dataset.

## Dataset
Titanic Dataset from Kaggle

## Tasks Performed
- Data Cleaning
- Missing Value Handling
- Exploratory Data Analysis
- Feature Engineering
- Logistic Regression Model Training
- Model Evaluation

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- scikit-learn

## Outcome
Successfully built and evaluated a classification model for survival prediction.
