# Yelp Review Classification

## Project Overview
Built a Natural Language Processing model to classify Yelp reviews based on customer ratings.

## Dataset
Yelp Review Dataset

## Tasks Performed
- Text Preprocessing
- Feature Extraction
- Data Vectorization
- Model Training
- Sentiment Classification

## Technologies Used
- Python
- Pandas
- NLTK
- scikit-learn

## Outcome
Successfully classified customer reviews using NLP and machine learning techniques.

## Learning Context

This project was completed as part of a hands-on Machine Learning course and focuses on applying Natural Language Processing techniques to real-world text data. The project helped strengthen skills in text preprocessing, feature extraction, sentiment analysis, classification modeling, and machine learning using Python and scikit-learn.
