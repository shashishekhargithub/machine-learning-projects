# Yelp Review Classification

## Project Overview
Built a Natural Language Processing model to classify Yelp reviews based on customer ratings.

## Dataset
Yelp Review Dataset

## Tasks Performed
- Text Preprocessing
- Feature Extraction
- Data Vectorization
- Model Training
- Sentiment Classification

## Technologies Used
- Python
- Pandas
- NLTK
- scikit-learn

## Outcome
Successfully classified customer reviews using NLP and machine learning techniques.
